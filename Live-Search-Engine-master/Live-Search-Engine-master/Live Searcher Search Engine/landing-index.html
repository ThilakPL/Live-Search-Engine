YouTube is a popular video-sharing platform that allows users to upload, share, and view videos. It was created in 2005 and is currently one of the largest websites in the world with billions of monthly active users. Unsplash


Unsplash is a platform that offers free high-quality images for personal and commercial use. With over 1 million images in its collection, it provides a wide range of images covering various topics such as nature, travel, food, people, and more. These images are contributed by talented photographers from around the world and can be used without the need for attribution. Unsplash images are an excellent source for those looking for visually appealing images for their website, blog, or social media posts.

GitHub




GitHub is a web-based platform for version control and collaboration that allows developers to store and manage their projects, track changes and communicate with others. It also acts as a repository for open-source software, allowing anyone to contribute to and use these projects. It has become an essential tool for developers and has transformed the way software is developed and shared.
